# ExplorerPro Suite

Advanced file explorer with privacy monitoring, preview panels, and sync capabilities.

## Features
- File browser with multi-tab support
- Privacy monitor for sensitive files
- Document preview (PDF, images, text)
- Sync manager
- Prompt launcher integration

## Installation
```bash
pip install -r requirements.txt
python src/main.py
```
